# Mari Docs

Mari is a prose-quality plugin for Claude Code. It catches weak writing during an edit, gives each finding a named rule, and helps Claude rewrite in the project's voice.

## What Mari does

- Checks prose with 170+ deterministic rules and 49 configurable word lists.
- Provides focused editorial workflows such as `deslop`, `tighten`, `clarify`, and `polish`.
- Enforces project terminology, style waivers, and zero-tolerance rules.
- Checks localization structure and documentation health.
- Maintains edit-notify rules and nudges for related-file obligations.
- Presents rules and project configuration in a local web console.
