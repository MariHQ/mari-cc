# Slash commands

| Command | Purpose |
|---|---|
| `/audit` | Report prose problems without editing. |
| `/deslop` | Remove canned, inflated, or mechanical phrasing. |
| `/tighten` | Cut repetition and unnecessary length. |
| `/clarify` | Resolve ambiguity and simplify difficult sentences. |
| `/sharpen` | Make key claims more forceful and specific. |
| `/understate` | Remove unnecessary explanation or restatement. |
| `/critique` | Evaluate structure, reasoning, and voice. |
| `/polish` | Run a final multi-category editorial pass. |
| `/draft` | Outline and write a new document in the project's voice. |

Each editing command reads project context, uses detector findings, preserves factual content, and reviews the result after editing.
