# cadence — fix sentence rhythm, flow, and length variation

Prose where every sentence is the same length reads like a metronome. Vary the rhythm.

## Flow
1. Run the detector; note `long-sentence` hits and stretches of uniform length.
2. Rework the rhythm:
   - break monotone runs — follow a long, clause-stacked sentence with a short one
   - split run-ons (`long-sentence`) and merge *accidental* choppiness — a deliberate fragment
     that lands is human rhythm, keep it
   - vary sentence openings; don't start five in a row the same way — and sentence-initial
     "And", "But", "So" are legitimate variety, not errors
   - thin out reflexive tricolons (`tricolon-overuse`) so the pattern stops being the tell
3. Read it aloud (mentally) — cadence is an ear problem. Preserve meaning; you're moving the
   joints, not the content.

Leans on: `long-sentence`, `tricolon-overuse`, sentence-length-variance signal.
