# Mari writing style

- Lead with the outcome.
- Use plain, direct sentences and specific verbs.
- Name mechanisms only when they help the reader act.
- Keep headings descriptive and sentence-cased.
- Prefer examples over abstract claims.
- Avoid canned transitions, inflated promises, and repeated conclusions.

## Terminology

| Use | Not |
|---|---|
| dataset | data set |
| word list | lexicon |
| finding | score event |
| repository config | global config |
| edit-notify rule | dependency graph |
